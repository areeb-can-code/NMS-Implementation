Network Scripting and Management

As a student of network scripting and management, I am thrilled to showcase my project on a web-based Network Management system that includes Fault, Configuration, Accounting, Performance, and Security Management features. This system is designed to be easy to use and navigate, and it is built using a range of technologies, including SNMP, Python scripting, and bash scripting, all within a CGI environment.

Key Features

  • Fault Management: This feature includes identifying link up and down events with a red or green color, showing the most recent 30 events in reverse order, and using a Python script to convert for the CGI environment.
  • Configuration Management: The Configuration Management page shows device configuration information and system information, provisions and updates system information, and enables or disables network interfaces.
  • Accounting Management: This page provides usage reports.
  • Performance Management: This page includes Throughput, System Load Report, and RTT Report.
  • Security Management: This page includes the intrusion detection report, showing IP addresses with more than 20 failures, and all syslog data.
